The data explored in this project comes from two separate sources:

-Firstly, a highly detailed csv file from Kaggle was accessed in which data covers over 5 million used and new car listings within the state of Illinois from June 2018 to June 2020. The data used from this source was used to provide general comparitive reference to the data analyzed from our second source.

-Secondly, a web-scraping algorithm was designed to retrieve data on new car listings from cargurus.com. The scraped data was restricted to the following zip codes and locations within a 50 mile radius of each one to cover the four main metroplexes in the state: 76012-Arlington, 78234-San Antonio, 78704-Austin, & 77033-Houston. From then, the data was filtered to only reflect the Ford Bronco Maverick, GMC Hummer EV, Chevrolet Blazer, Toyota Tundra, and 3 variations of the Jeep Wrangler.
